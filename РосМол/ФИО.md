## Название проекта
Система учёта поставок молока для АО ГК «Российское молоко»

## Описание проекта
Приложение разработано на языке C# с использованием WPF.  
Предназначено для учёта поставщиков и поставок молока.

## Функции
- добавление поставок;
- просмотр данных;
- работа с базой данных.
